# Process Scheduling

A C++ implementation of various CPU Process Scheduling algorithms used in Operating Systems. This project demonstrates how different scheduling techniques allocate CPU time to processes and compares their performance based on waiting time and turnaround time.

## Features

- CPU Scheduling Simulation
- Calculates Waiting Time
- Calculates Turnaround Time
- Displays Average Waiting Time
- Displays Average Turnaround Time
- Console-based implementation
- Written in C++

## Scheduling Algorithm


Example:
- First Come First Serve (FCFS)
- Shortest Job First (SJF)
- Priority Scheduling
- Round Robin



## Technologies Used

- C++
- Standard Template Library (STL)
